# EQ - Embedded Queueing

EQ is a little framework to queue tasks between threads in a persistent way.

## Installation

Add this line to your application's Gemfile:

    gem 'eq'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install eq
